# 🛒 E-Commerce Website

A full-stack **E-Commerce Web Application** designed to provide users with a smooth online shopping experience.
The system supports product browsing, cart management, order processing, and admin control.

---

## 📌 Project Overview

This project simulates a real-world e-commerce platform where:

* Users can browse and purchase products
* Admins can manage products and orders
* Data is stored securely in a database
* Backend logic is exposed through REST APIs

---

## ✨ Features

### User Features

* User registration and login
* Browse products
* Search and filter products
* Add products to cart
* Place orders
* View order history

### Admin Features

* Admin authentication
* Add, update, and delete products
* Manage product categories
* View and manage orders

---

## 🏗️ Architecture

* Frontend handles user interface
* Backend processes business logic
* Database stores application data
* Communication via REST APIs

---

## 🧰 Tech Stack

* **Frontend:** HTML, CSS, JavaScript,React JS
* **Backend:** Java (Spring Boot)
* **Database:** PostgreSql
* **Version Control:** Git & GitHub
